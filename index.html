Request desk

Заявка открывает готовое письмо менеджеру

Без backend нельзя безопасно хранить Telegram bot token или CRM-secret в публичном репозитории. Поэтому текущая production-safe версия использует `mailto:`: форма собирает данные и открывает email-клиент с готовым письмом.
