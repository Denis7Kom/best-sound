Privacy / prototype

Политика обработки персональных данных

Эта страница подготовлена для статического прототипа сайта BEST SOUND. Она не заменяет юридическую политику, подготовленную юристом, но закрывает базовый пользовательский сценарий: посетитель понимает, какие данные он передаёт через форму заявки.

Какие данные собирает форма

Форма заявки может содержать:

  • имя;
  • телефон;
  • email;
  • тип клиента;
  • город и дату мероприятия;
  • бюджет;
  • описание задачи.

Как работает отправка в этой версии

В текущем статическом прототипе данные не сохраняются на сервере сайта. После отправки формы браузер открывает email-клиент пользователя с подготовленным текстом письма. Само письмо отправляется только после действия пользователя в его почтовом приложении.

Для чего используются данные

Данные используются только для связи по заявке, обсуждения мероприятия, уточнения условий, подготовки предложения и дальнейшей коммуникации по проекту.

Что нужно для production-версии

Для реального коммерческого запуска рекомендуется заменить эту страницу на юридически проверенную политику, добавить реквизиты оператора данных, срок хранения, порядок удаления данных и полноценный серверный канал отправки заявок.
