# BEST SOUND — Concert Agency Website Prototype

Современный статический прототип сайта для концертного агентства **BEST SOUND**.

Проект сделан как быстрый, но презентабельный frontend-прототип: без базы данных, без backend и без внешних библиотек. При этом структура уже подготовлена для дальнейшего развития: данные вынесены отдельно, интерфейс рендерится через JS, стили отделены от HTML.

## Что реализовано

- Premium-tech визуальный стиль.
- Hero-блок без эквалайзера и кривых декоративных волн.
- SVG-сцена с прожекторами и экраном BEST SOUND.
- Bento-секция для современной презентации агентства.
- Интерактивная афиша с фильтрами.
- Quick-view модальное окно концерта.
- Данные концертов, услуг и маршрута вынесены в `src/data.js`.
- SVG-карта тура с городами.
- Улучшенная форма заявки:
  - имя;
  - телефон;
  - email;
  - тип клиента;
  - город;
  - дата;
  - бюджет;
  - описание задачи;
  - согласие на обработку данных.
- SEO / Open Graph / favicon.
- `robots.txt` и `sitemap.xml`.
- Базовая accessibility-поддержка:
  - skip-link;
  - `focus-visible`;
  - `prefers-reduced-motion`;
  - закрытие меню/модалки через `Escape`;
  - aria-атрибуты.

## Как запустить

Вариант 1 — открыть `index.html` напрямую в браузере.

Вариант 2 — запустить локальный статический сервер:

```bash
python -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```

## Что важно понимать

Это не production-сайт и не финальная коммерческая версия. Это качественный frontend-прототип, который можно показывать как направление редизайна.
