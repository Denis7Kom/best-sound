# BEST SOUND — Concert Agency Website

Статический сайт для концертного агентства **BEST SOUND**.

Текущая версия сделана как улучшенная и более современная версия официального сайта: без лишней выдуманной подачи, без декоративного мусора и без технических пояснений в интерфейсе. Основная структура соответствует текущему сайту BEST SOUND: концерты, фотогалерея, видео и контакты.

## Языки

- `index.html` — основная русская версия.
- `lang-en.html` — английская версия.
- `lang-it.html` — итальянская версия.

## Что реализовано

- Главная русская страница с основной структурой сайта.
- Дополнительные страницы на английском и итальянском языках.
- Переключатель языков `RU / EN / IT`.
- Чистый текстовый wordmark в шапке: `Concert Agency / BEST SOUND`.
- Разделы:
  - Concerts / Концерты;
  - Photo Gallery / Фотогалерея;
  - Video / Видео;
  - Contacts / Контакты.
- Блок с официальными тезисами:
  - BEST REPUTATION;
  - OVER 20 YEARS OF EXPERIENCE;
  - HUNDREDS OF SUCCESSFUL EVENTS;
  - THOUSANDS OF SATISFIED VIEWERS;
  - COOPERATION WITH WORLD STARS;
  - WIDE GEOGRAPHY OF CONCERTS.
- Опубликованные артисты и даты:
  - Alessandro Safina;
  - Federico Paciotti;
  - Joe Lynn Turner.
- Контактный блок с WhatsApp / Telegram и email.
- `privacy.html` — базовая страница политики обработки данных.
- `sitemap.xml` с русской, английской, итальянской страницами и privacy page.
- `robots.txt`.
- Адаптивные layout-fixes в `src/official.css`.

## Как запустить локально

Можно открыть `index.html` напрямую в браузере.

Лучше запустить локальный сервер:

```bash
python -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```

Проверить языки:

```text
http://localhost:8000/
http://localhost:8000/lang-en.html
http://localhost:8000/lang-it.html
```

## GitHub Pages

Через настройки репозитория:

```text
Settings → Pages → Build and deployment → Deploy from a branch → main → /root
```

После включения Pages сайт должен быть доступен по адресу:

```text
https://denis7kom.github.io/best-sound/
```
